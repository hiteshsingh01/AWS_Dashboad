## üîê Required IAM Permissions

Your IAM user should have the following EC2 permissions:

* `ec2:RunInstances`
* `ec2:TerminateInstances`
* `ec2:StartInstances`
* `ec2:StopInstances`
* `ec2:DescribeInstances`
* `ec2:CreateKeyPair`
* `ec2:CreateSecurityGroup`
* `ec2:AuthorizeSecurityGroupIngress`
* `ec2:CreateTags`
